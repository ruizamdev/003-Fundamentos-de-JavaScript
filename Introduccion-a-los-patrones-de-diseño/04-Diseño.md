# Fase de Diseño de Software

Al momento de construir aplicaciones hay dos perspectivas diferentes del diseño:

- Equipo de UX/UI
- Equipo de Ingeniería

## Algunos elementos de un buen diseño de Software

1. Modularidad
2. Tolerancia a fallos
3. Robustez
4. Seguridad
5. Usabilidad
6. Reusabilidad
7. Extensibilidad
